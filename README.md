### 1. Install Dependencies

```bash
npm init -y
```

```bash
npm install express mysql body-parser cors
```

### 2. Structure

```bash
crud-app/
│
├── app.js         # Node.js API
├── public/
│   └── index.html # Frontend HTML + JS
```

### 3. Run the App

```bash
node app.js
```

<b>Visit:</b> <a>http://localhost:3000</a>
